## 📌 <a id="test_cases"></a>测试用例快速检查清单

### 功能测试

**NFC 读取**
- [ ] 点击"读取"按钮后状态正确
- [ ] 靠近 NFC 标签能成功读取
- [ ] 读取的数据正确保存到数据库
- [ ] UI 显示正确的 NFCID

**NFC 写入**
- [ ] 输入卡号和车号后点击"写入"
- [ ] 靠近 NFC 标签能成功写入
- [ ] 写入的数据正确保存到数据库
- [ ] 标签可以被正确读回

**表格显示**
- [ ] 刚启动时表格为空
- [ ] 读取/写入后表格实时更新
- [ ] 列表项显示所有必要信息
- [ ] 滚动列表流畅

**蓝牙打印**
- [ ] 能连接到已配对的蓝牙打印机
- [ ] 点击"打印"后数据正确发送
- [ ] 打印机正确打印数据
- [ ] 连接异常时有提示信息

**数据库**
- [ ] 新数据能正确保存
- [ ] 能读取之前保存的数据
- [ ] 数据格式正确

### 权限测试
- [ ] NFC 权限正确请求
- [ ] 蓝牙权限正确请求
- [ ] 拒绝权限时应用正确处理
- [ ] 授予权限后功能正常

### 兼容性测试
- [ ] 在 Android 5.0 设备上运行
- [ ] 在 Android 13 设备上运行
- [ ] 在不同 NFC 芯片的标签上测试
- [ ] 在不同蓝牙打印机上测试

### 性能测试
- [ ] 启动时间 < 3 秒
- [ ] 表格滚动帧率 > 50 FPS
- [ ] 内存占用正常
- [ ] 蓝牙连接响应 < 2 秒

### 异常处理测试
- [ ] 设备不支持 NFC 时的提示
- [ ] 没有启用 NFC 时的提示
- [ ] 蓝牙连接失败时的提示
- [ ] 数据库错误时的提示
